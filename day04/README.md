# Advent of Code 2024 – Day 4: Ceres Search

## Overview

The challenge consists of searching a character grid to find specific patterns.

- **Part 1:** Count every occurrence of the word **"XMAS"** in all eight possible directions.
- **Part 2:** Count every **X-MAS** pattern where two diagonal `"MAS"` sequences intersect at the letter `'A'`.

## Results

| Part | Answer |
|------|--------|
| Part 1 | **2507** |
| Part 2 | **1969** |
