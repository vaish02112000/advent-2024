# Day 5 - Print Queue

## Overview

This solution validates and corrects page update sequences using a set of page ordering rules.

* **Part 1:** Identifies updates that already satisfy all ordering rules and computes the sum of their middle page numbers.
* **Part 2:** Corrects the invalid updates using Topological Sorting (Kahn's Algorithm) and computes the sum of the middle page numbers of the corrected updates.

## Output

The program prints the answers for both Part 1 and Part 2.

Part1-4609
Part2-5723
