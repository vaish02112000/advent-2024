# Day 1 - Historian Hysteria

This repository contains my Python solution for **Advent of Code 2024 – Day 1**.

## Overview

The solution implements both parts of the challenge:

- **Part 1:** Calculates the total distance between two lists by sorting them independently and summing the absolute differences between corresponding elements.
- **Part 2:** Computes the similarity score by counting the occurrences of each value in the right list and multiplying each value in the left list by its frequency.

## Output
Part 1: 1189304
Part 2: 24349736
