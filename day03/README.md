# Day 3 - Mull It Over

## Overview

This solution processes corrupted memory instructions to identify and evaluate valid multiplication operations.

- **Part 1:** Extracts all valid `mul(X,Y)` instructions and computes the sum of their products.
- **Part 2:** Extends the parser to handle `do()` and `don't()` instructions, enabling or disabling multiplication while scanning the input.
