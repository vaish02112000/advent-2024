# Day 6 - Guard Gallivant

## Overview

This solution simulates the guard's patrol through the laboratory according to the movement rules.

* **Part 1:** Simulates the guard's movement and counts the number of distinct positions visited before leaving the map.
* **Part 2:** Places a single obstruction at every possible visited position (except the starting position), detects whether it causes the guard to enter a loop, and counts all such valid positions.

## Output

The program prints the answers for both Part 1 and Part 2.

Part1- 4789

Part2- 1304
